# Personal Portfolio

This repository powers my personal portfolio and resume site at [supernet.tech](https://www.supernet.tech).

The site started as an older Bootstrap and AngularJS experiment from earlier in my career. I rebuilt it as a static Next.js and Tailwind CSS site to better reflect how I currently approach frontend development: pragmatic tooling, authored content, intentional UI, and as little runtime complexity as the problem allows.

## Why This Exists

The live site is meant to be a concise introduction to my work, engineering background, and current software interests. This repo is the source for that site and a small behind-the-scenes look at the choices behind it.

## Design Goals

- Keep the site fast, static, and easy to host.
- Use authored pages instead of a generic CMS-like renderer.
- Preserve a bit of personality in the intro experience.
- Make each project page flexible enough to tell its own story.
- Avoid unnecessary backend or server complexity.

## Tech Choices

- Next.js
- TypeScript
- Tailwind CSS
- Static export

## Notes

This is intentionally not a full product application. It is a personal site, so the implementation favors clarity, maintainability, and visual polish over generalized abstractions.
